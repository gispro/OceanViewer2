Приложение OceanViewer предназначено для работы с пространственными данными Единой системы информации об обстановке в Мировом океане (ЕСИМО) в среде Интранет/Интернет. Приложение не требует установки специального программного обеспечения или данных на локальном компьютере и запускается в любом из Интернет браузеров: Internet Explorer 8+, Mozilla Firefox 3+, Opera, Google Chrome, Safari и др.

OceanViewer позволяет просматривать, добавлять, редактировать и анализировать картографические и пользовательские данные ЭМА, изменять по своему усмотрению набор просматриваемых данных и сохранять изменения в виде пользовательских карт для последующей работы.

Версия: OceanViewer 0.4.4
